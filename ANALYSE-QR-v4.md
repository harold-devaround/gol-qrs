### Analyse : éléments des CP NON couverts par les tuiles (→ probablement chasse)

Les tuiles sont toutes des **îles fantastiques** vues de loin. Elles ne représentent aucun objet spécifique des CP (pas de sudoku, pas d'horloge, pas de bocal, pas de partition). Le mécanisme du jeu est de **pointer** un élément visuel de la CP qui **évoque** une tuile (ex : pointer un bateau sur CP7 pour faire deviner la tuile « bataille navale »).

**Conséquence** : les objets « pointables » sur les CP (animaux, objets reconnaissables, scènes) servent au **jeu**. Les éléments **non pointables** ou **non représentés par les tuiles** (cryptos, codes, symboles, texte) sont pour la **chasse**.

Éléments clairement **chasse uniquement** (non pointables pour les tuiles) :
- Les **grilles de chiffres** (sudoku CP1, carton CP9)
- Les **symboles triangulaires** sur les bocaux (CP2)
- Les **runes MHF** du tapis (CP2)
- Les **heures exactes** des horloges (CP4)
- Le **post-it jaune** / IS1 (CP4)
- Les **symboles gravés** sur la barrière (CP5)
- Les **positions exactes des étoiles** dans le ciel (CP6)
- Les **motifs des fanions/pavillons** (CP7)
- La **partition musicale** (CP8)
- Les **nombres sur le carton** (CP9)
- Les **positions des mouches** sur la toile (CP9)
- Le **QR code** au verso de CP5
- Le **texte OE BIIGIBDEH** (CP2)

### Nouveau : Correction Room 144 (room_precision_horloges.png)

Image trouvée dans la Room 144 montrant un **diagramme de correction** :
- **Horloge gauche** (sous un **☀ Soleil**) montrant ~2h50/3h50 → avec flèche → **Horloge droite** (sous un **☽ Lune**)
- **Soleil = PM** (ajouter 12h), **Lune = AM** (lire tel quel)
- Les deux cadrans utilisent la notation horlogère **IIII** (pas IV)

#### Mécanisme probable pour la 4ème horloge :
Le problème : la lecture naïve donne le 4ème caractère = **o** (15 → O), donnant PAR**o**S.
La correction donne = **I** (9 → I), donnant PAR**I**S.

| Scénario | Mécanisme | Résultat |
|----------|-----------|----------|
| A — Inversion Soleil/Lune | L'horloge 4 porte un soleil mais doit être lue AM (lune) → 9h = I | PARIS ✓ |
| B — Inversion aiguille | On lit la grande aiguille au lieu de la petite (#3 : « la pointe est parfois sur la petite et parfois sur la grande ») → position :09 → I | PARIS ✓ |
| C — Conversion PM spéciale | 12h35 PM = 00h35 → 35 mod 26 = 9 → I | PARIS ✓ |
| D — Notation 24h | 24:35 → interprétation donnant I au lieu de O | PARIS ✓ |
